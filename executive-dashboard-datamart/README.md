# Executive Dashboard Data Mart

This project demonstrates a unified KPI-ready data mart used to support executive dashboards and business performance reporting.

## Business Use Case
Leadership teams need one place to view customer, revenue, booking, and usage metrics together. This project models an executive reporting layer that combines multiple subject areas.

## Files
- `dim_customers.sql`
- `fct_bookings.sql`
- `fct_revenue.sql`
- `fct_usage.sql`
- `mart_executive_kpis.sql`

## What This Demonstrates
- dimensional modeling
- KPI layer design
- multi-domain reporting
- executive-level analytics architecture
