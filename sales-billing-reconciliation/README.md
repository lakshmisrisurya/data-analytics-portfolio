# Sales to Billing Reconciliation

This project demonstrates a cross-system reconciliation model that identifies mismatches between CRM opportunities, contracts, and invoices.

## Use Cases

- identify missing contracts
- identify contract to invoice gaps
- highlight booking mismatches
- support finance and rev ops review

- `stg_opportunities.sql`
