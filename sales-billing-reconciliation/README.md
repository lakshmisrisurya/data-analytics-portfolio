# Sales to Billing Reconciliation

This project demonstrates a reconciliation model that compares sales bookings, contract values, and invoice amounts across systems.

## Business Use Case
Organizations often face reporting gaps between what sales closed, what was contracted, and what was invoiced. This project helps identify mismatches and missing process handoffs.

## Files
- `stg_opportunities.sql`
- `stg_contracts.sql`
- `stg_invoices.sql`
- `fct_reconciliation.sql`

## What This Demonstrates
- cross-system reconciliation
- finance and rev ops reporting support
- gap detection logic
- operational reporting design
