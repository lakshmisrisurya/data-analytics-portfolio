# Marketing Performance Analytics

This project demonstrates marketing funnel and campaign performance analytics using lead, opportunity, and spend data.

## Business Use Case
Marketing and GTM teams need visibility into how campaigns drive pipeline, conversion, and revenue. This project models lead funnel performance and campaign ROI across channels.

## Metrics Included
- Leads
- MQLs
- SQLs
- Opportunities Created
- Pipeline Generated
- Closed Won Revenue
- Lead to MQL Conversion
- MQL to SQL Conversion
- SQL to Opportunity Conversion
- Opportunity to Closed Won Conversion
- Cost per Lead
- Cost per MQL
- Cost per Opportunity
- Return on Ad Spend
- Customer Acquisition Cost

## Files
- `stg_marketing_leads.sql`
- `stg_campaign_spend.sql`
- `fct_marketing_funnel.sql`
- `fct_campaign_roi.sql`

## What This Demonstrates
- funnel analytics
- campaign performance reporting
- spend efficiency analysis
- revenue attribution support
