# Product Usage Analytics

This project transforms event-level product activity into account-level and feature-level usage metrics.

## Business Use Case
Product, customer success, and leadership teams need visibility into engagement and adoption trends. This project models usage activity to support adoption reporting and product decision-making.

## Files
- `stg_events.sql`
- `fct_daily_usage.sql`
- `fct_feature_adoption.sql`
- `usage_summary.py`

## What This Demonstrates
- event data modeling
- daily active usage logic
- feature adoption analysis
- product analytics reporting
